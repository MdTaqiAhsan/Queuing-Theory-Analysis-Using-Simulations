# Queuing-Theory-Analysis-Using-Simulations
In this project, we use realistic arrival and service assumptions to model three important queues:
1. Toll booth with one server – modeled as an M/M/1 queue
2. Toll booth with two servers – modeled as an M/M/2 queue
3. Caf´e with limited queue capacity – modeled as an M/M/1/K queue
